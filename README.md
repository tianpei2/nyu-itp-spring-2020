# Marsbot

## Running Locally

Make sure you have Python installed. To push to Heroku, you'll need to install the [Heroku CLI](https://devcenter.heroku.com/articles/heroku-cli)

## Deploy!
For an existing Heroku app,  
`heroku login`  
`heroku git:remote -a [app name]`
